# 🎓 Student Record Management System

A simple Student Record Management System built using **Python**, **Streamlit**, and **SQLite**.

## Features
- Admin Login
- Dashboard
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Export CSV
- Export Excel

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- OpenPyXL

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Deploy
1. Push this project to GitHub.
2. Connect the repository to Streamlit Cloud.
3. Deploy `app.py`.
